## 📌 Visão Geral do Projeto

A **Shalom Films** é uma empresa especializada em instalação de películas em vidros residenciais e comerciais. O objetivo deste projeto foi desenvolver uma vitrine digital que apresentasse os benefícios técnicos do produto (proteção UV, redução de calor e privacidade) de forma visual e direta.

A aplicação foi construída priorizando a **velocidade de carregamento** e a **facilidade de contato**, resultando em uma interface intuitiva que guia o usuário desde a descoberta do benefício até o fechamento do orçamento.

---

## 🛠️ Engenharia de Software & Escolhas Técnicas

Para este projeto, apliquei as melhores práticas de desenvolvimento Front-end:

* **Arquitetura Baseada em Componentes:** Desenvolvimento de UI modular utilizando React, permitindo fácil manutenção e reaproveitamento de código em seções como a galeria de serviços e depoimentos.
* **Otimização de Performance:** Implementação de estratégias de *Lazy Loading* para imagens pesadas, garantindo que a página seja carregada rapidamente mesmo em conexões móveis limitadas.
* **Design Responsivo Dinâmico:** Uso de **Tailwind CSS** para garantir que a experiência de navegação seja idêntica e funcional em telas de 4 polegadas até monitores 4K.
* **SEO Local:** Estruturação de tags HTML para otimizar o rankeamento orgânico da empresa em buscas locais por serviços de películas e vidros.
